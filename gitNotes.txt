To download SDL, just download it from the website
    https://glusoft.com/sdl2-tutorials/install-sdl-linux/

Creating a new branch
git checkout -b [branch_name]
git add .
git commit -m "Your commit message here"
git push origin opcode
git checkout main
git merge [branch_name]

Switch to existing branch
git checkout [branch_name]
git pull
git add .
git commit -m "Your commit message here"
git push origin opcode
git checkout main
git merge [branch_name]
